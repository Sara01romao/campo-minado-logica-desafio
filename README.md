## 💻  Sobre o Projeto
Projeto desenvolvido com foco em lógica de programação, recriando o funcionamento do jogo Campo Minado.

## 🔧 Etapas

- Definir o tamanho da matriz (campo do jogo)
- Calcular a quantidade de minas com base na dificuldade
- Gerar posições aleatórias para as minas
- Criar a matriz do jogo
- Inserir as minas no mapa
- Calcular a quantidade de minas ao redor de cada posição
- Implementar a lógica de abertura das células
- Criar a verificação de vitória e derrota

<br>

## :rocket: Tecnologias Usadas

```
JS → lógica do jogo (matriz, minas, regras)
jQuery → manipulação do DOM
HTML/CSS → interface
```
